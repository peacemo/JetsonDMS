# JetsonDMS 

## Introduction
JetsonDMS (Jetson Driver Monitoring System) is an embedded vision system built with:

- Jetson computer vision 
- driver farigue detection algorithm
- real time CAN communication

## Hardware

- Jetson Orin Nano Super 8GB Devkit
- Waveshare (微雪) SN65HVD230 CAN Module
- GCAN (广成) USB-to-CAN-2II 
- 800M IMX219 CSI camera

  - TensorRT
