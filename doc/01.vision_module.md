# Vision Module Design

## Functionality

1. Camera capturing
2. Image reading (raw or file read?)
4. Image prepprocessing
3. AI inferencing by tensorrt accelaration
4. Status responsing

## Software

1. OpenCV C++ 
2. TensorRT C++

## Module Designs

### Image capturing

input: no input

output: raw image data (or image file name)

procedure:

- check if camera online
- capture a frame by camera toolkit
- return raw frame data

### Image reading

input: raw frame data

output: opencv image data structure

procedure:

- read raw frame data by opencv into an image
- return the image data structure

### Image preprocessing

input: opencv image

output: opencv image

procedure: 

- apply basic image preprocessing
- return processed image 

### AI model inferencing

input: opencv image

output: C++ detect status

procedure:

- convert opencv image to tensorrt vector
- do model infer to get raw prediction data
- process model output 
- return C++ status

### C detection API

input: 

output: C detect status

procedure:

- call full C++ vision procedure
- convert C++ status to C status
- return C status
